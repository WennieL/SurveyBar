
# Survey Bar 📊 (演示版)

> **⚠️ 演示版 (DEMO) 注意事項**
> 
> 本應用程式僅為 **概念驗證 (POC) / 演示用途**。
> *   **無涉及真實金錢**：「錢包」功能使用模擬交易，不會實際扣款。
> *   **資料儲存於本地**：所有的問卷、點數和使用者資料皆儲存在您瀏覽器的 `localStorage` 中。清除快取將會重置應用程式。
> *   **問卷連結**：目前顯示的連結多為範例連結 (如 Google Forms)。

## 簡介

**Survey Bar** 是一個社群驅動的問卷互助市集。核心概念非常簡單：**填問卷，換回饋 (Give Feedback to Get Feedback)**。

*   **設計師與創業者**：為您的點子快速獲得真實誠懇的回饋。
*   **學生與研究人員**：輕鬆找到學術研究的受訪者。
*   **社群成員**：透過幫助他人填寫問卷賺取點數，再用點數推廣自己的專案。

## 運作方式

### 1. 賺取點數 (經濟體系)
為了確保高品質的回覆，平台採用點數系統運作。
*   前往 **瀏覽問卷 (Browse)** 分頁。
*   透過興趣篩選 (例如：*產品發布*、*學術研究*)。
*   點擊 **填寫問卷 (Take Survey)**。系統將引導您至外部問卷連結 (Google Forms, Typeform 等)。
*   誠實完成問卷。
*   返回 Survey Bar 並點擊 **「我已完成」** 以領取點數。
    *   *標準獎勵*：1 點
    *   *急件/長篇問卷*：最高 3 點

### 2. 發布您的問卷
當您累積足夠點數後，即可為自己的專案尋找受訪者。
*   點擊 **發布問卷 (Post Survey)**。
*   貼上您的問卷連結 (Google Forms, SurveyMonkey 等)。
*   設定 **目標受眾** (地點、年齡、語言)。
*   **費用**：
    *   10 點 (基本刊登費)
    *   額外點數可用於「推廣置頂」或設定更高的填答獎勵。
*   您的問卷將立即顯示在列表中！

### 3. 推廣與數據
*   **推廣 (Boost)**：使用點數將您的問卷置頂，以獲得更快速的回覆。
*   **我的問卷**：追蹤有多少人點擊並參與了您的問卷。

### 4. 等待名單 (Google Apps Script 整合)
此演示版包含一個可運作的 Google Sheets 後端整合。
*   點擊 **「加入搶先體驗 (Join Early Access)」**。
*   輸入您的姓名與 Email。
*   資料將會即時傳送到後端的 Google Sheet！

## 安全性說明 (Google Apps Script)
本應用程式使用前端的 Google Apps Script 網址 (`constants.ts` 中的 `GOOGLE_SCRIPT_URL`) 來傳送等待名單資料。
*   **風險**：由於此網址直接寫在前端程式碼中，任何人都可以看到並可能向您的試算表發送垃圾資料。
*   **安全性**：使用者**無法**透過此網址讀取或刪除您的試算表資料（除非腳本另有設定），它僅作為寫入用途。
*   **正式環境建議**：若是正式運作的應用程式，建議透過後端伺服器轉發請求，或是在 Google Apps Script 中加入驗證碼或流量限制機制。

## 技術設置 (開發者指南)

本專案使用 **React**, **TypeScript**, 和 **Tailwind CSS** 建構。

### 前置需求
*   Node.js (v16+)
*   npm 或 yarn

### 安裝步驟

1.  複製此儲存庫 (Clone repo)。
2.  安裝依賴套件：
    ```bash
    npm install
    ```
3.  啟動開發伺服器：
    ```bash
    npm start
    ```

### 設定後端

本應用程式使用 Google Apps Script Web App URL 作為等待名單的功能後端。設定位於 `constants.ts` 檔案中。

若要設定您自己的後端：
1.  建立一個 Google Sheet。
2.  前往 **擴充功能 > Apps Script**。
3.  部署一個 `doPost(e)` 函式來接收 POST 請求。
4.  將網址更新至 `constants.ts` 中的 `GOOGLE_SCRIPT_URL`。

## 專案結構

*   `App.tsx`: 應用程式主控制器。
*   `components/`: UI 元件 (導航列、問卷卡片、模態視窗)。
*   `services/storageService.ts`: 處理資料持久化 (使用 `localStorage` 模擬資料庫)。
*   `translations.ts`: 多語系支援 (英文 & 繁體中文)。
*   `types.ts`: TypeScript 型別定義 (使用者與問卷)。
